# Skill: Script Writing for Instagram Reels & Carousels

## Kullanım Alanı
Bu skill, competitor analizi sonuçlarını Nova'nın sesine dönüştürülmüş tam video senaryolarına çevirir.

---

## Reel Script Yapısı (60–90 saniye)

```
[HOOK — 0–3 sn]
Görsel veya sözlü kanca. İzleyiciyi "Bu benim için!" hissettir.

[CONTEXT — 3–10 sn]
Neden bu konuyu konuşuyoruz? Kısa bağlam, acı noktasını doğrula.

[CORE — 10–50 sn]
3 adım / 3 nokta / 3 sebep formatı. Her nokta net, kısa, eyleme geçirilebilir.

[CTA — 50–60 sn]
Tek bir net çağrı: kaydet / yorum yaz / takip et / soru sor.
```

---

## Carousel Script Yapısı (6–10 Slide)

| Slide | İçerik |
|-------|--------|
| **1 (Cover)** | Hook — Tek güçlü cümle. Büyük font, minimal tasarım. |
| **2** | Problemi tanımla — "Çoğu [kitle] şunu yapar..." |
| **3–7** | Her slide = 1 nokta. Başlık + 2 cümle açıklama. |
| **Son-1** | Özet veya "Altın Kural" |
| **Son** | CTA — "Kaydet", "Bizi takip et", veya soru |

---

## Nova'nın Script Ses Kılavuzu

### Yapı Tercihleri
- **Kısa cümleler:** Maksimum 12 kelime per cümle
- **Aktif fiiller:** "Yapın", "Kullanın", "Deneyin" — pasif cümlelerden kaçın
- **Sayılar:** Rakam kullan — "üç şey" yerine "3 şey"
- **Örnekler:** Her soyut ifadenin yanında somut bir örnek

### Duygu Tonu Haritası
| Konu | Ton |
|------|-----|
| Kariyer / Gelecek | Güçlendirici, umut verici |
| Araç / Teknik | Meraklı, pratik, keşifçi |
| Hata / Ders | Dürüst, alçakgönüllü, öğretici |
| Vlog / Günlük | Sıcak, samimi, ilişkilendirilebilir |

---

## Competitor → Script Dönüşüm Protokolü

1. **Konu al** — Rakibin en çok etkileşim alan postunun konusunu çıkar
2. **Duygusal tetikleyici bul** — Post hangi duyguya dokunuyor? (merak, FOMO, ilham, rahatlama)
3. **Nova çerçevesi** — Aynı duyguyu kendi deneyiminle yeniden kur
4. **3 nokta oluştur** — Öğretilebilir, eyleme geçirilebilir, özgün
5. **CTA seç** — O haftanın hedefine göre (takipçi artışı → soru sor; etkileşim → kaydet)

## Örnek Script (Rakip: thedvlprl, Konu: Vibe Coding)

```
[HOOK]
Görselde: Cursor AI'da hızla kod üretilen ekran
"6 ay önce günde 8 saat kod yazıyordum. Şimdi 2 saatte bitiriyorum."

[CONTEXT]
"Değişen şey ne biliyor musunuz? Araç değil — yaklaşım."

[CORE]
"Şimdi size Cursor AI ile nasıl çalıştığımı gösteriyorum:
1. Önce YAP — Cursor'a tam olarak ne istediğimi söylüyorum, ne nasıl değil
2. Anla — Üretilen kodu TAB'a basmadan önce mutlaka okuyorum  
3. Sahiplen — Her dosyayı kendi stilime uyarlıyorum"

[CTA]
"Siz vibe coding kullanıyor musunuz? Yorumlarda buluşalım."
```

## Script Kalite Kriterleri

✅ **Her script için:**
- [ ] Hook ilk 3 saniyede izleyiciyi tutuyor mu?
- [ ] Hiçbir nokta 2 cümleden fazla mı uzun? (Reels için)
- [ ] CTA tek ve net mi?
- [ ] Script sesli okunduğunda doğal geliyor mu?
- [ ] Nova'nın kişisel deneyimiyle mi bağlanıyor?
